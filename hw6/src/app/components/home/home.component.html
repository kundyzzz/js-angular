Welcome to University Portal

This small demo shows a list of universities, server-side search, and detail pages. Use the navigation above to explore the app.

Browse Universities About this project

What you can try

  • Search universities (server-side)
  • Open details page for each university
  • Use URL param ?q= to share search
  • Filter results dynamically
  • Learn more about each university
  • Easy navigation between pages
  • Responsive design for mobile and desktop

This project is a demo of Angular components, routing, service calls, and dynamic content rendering. You can explore how data flows from the server to the UI and practice working with API-driven apps.

Future improvements could include user authentication, saving favorite universities, and adding a dashboard for registered users.
